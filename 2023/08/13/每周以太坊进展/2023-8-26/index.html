2023-8-26

发表于 2023-08-13 更新于 2023-10-27 分类于 每周以太坊进展 阅读次数：
本文字数： 1.7k 阅读时长 ≈ 2 分钟

Dencun (Cancun + Deneb) 升级

  • 最新的所有核心开发人员 - 执行 (ACDE)视频会议。Christine Kim的记录：

    • Dencun测试概述
    • devnet 9 和升级公共测试网的时间表将在下次 ACDE 电话会议上讨论

  • KZG 仪式圆满结束，共 141,416 贡献者，这是有史以来规模最大的仪式

Layer 1

  • Holešky 测试网（替代Goerli）
    • 创世文件已生成，于 9 月 15 日启动，拥有 146 万个验证者
    • ACDC 同意将 Shapella 升级的 epoch 时间从 10 调整为256，验证者踢出余额调整为 28 hETH ，并支持压缩创世状态.
  • Dapplion：Whisk (SSLE) 昂贵的职责发现（duty discovery）可以用验证者的公钥来解决

给质押者

  • 基于来自大型质押池和实体的自我报告数据，绝大多数执行层客户端仍然是Geth
  • Siren v1（Lighthouse 的验证器管理 UI）：监控节点和验证器健康状况

客户端版本

  • 共识层：
    • Nimbus v23.8.0：性能和稳定性改进
    • Prysm v4.0.8：减少内存使用
  • 执行层：
    • Besu v23.7.1：EVM 工具和 RPC 跟踪模块的性能和稳定性改进、更新依赖项和扩展功能
    • Reth v0.1.0-alpha.7：修复爬虫到达节点问题、RPC 和 txpool 以及帐户的 eth_getProof 初步支持

研究

  • Casper的超级确定性 确认规则，可以使用Beacon API 在客户端/钱包中实现
  • 验证器证明：分布式哈希表参与者的匿名凭证方案

Layer 2

  • RISC Zero Zeth：基于 revm 的 zk 区块生成证明器（zkEVM）

  • Base

    使用升级多重签名 2/2 （Optimism Foundation & Base）

    • Coinbase 将向 Optimism Collective 贡献至少 2.5% 的 Base 排序器收入，用于 RetroPGF 和生态系统项目
    • Optimism 在 6 年内向 Coinbase 授予约 1.18 亿个 OP 代币（占供应量的 2.7%）

EIP

  • 更新了EIP 编辑手册(EIP5069)
  • ERC（应用层）：
    • ERC7502：可自交易的NFT
    • ERC7503：零知识虫洞
    • ERC7504：动态合约（一对多代理合约）

开发者资料

  • Foundry:
    • Vulcan v0.2.0（开发框架）：添加gas估算和部署地址计算
    • 使用 jwt 秘密的 Cast rpc允许Engine API 调用执行层
  • 使用 ScaffoldETH-2 和 The Graph 的全栈 dapp workshop
  • Ape 框架代码覆盖测试指南
  • Ape-hackathon-kit：在 Ape 框架中开发合约，在 Next.js、Tailwind、RainbowKit 和 wagmi 中开发 Web 应用程序
  • WhatsABI v0.8.0（字节码推测 ABI）：使用 ethers.js v5/6 和 viem 进行测试，包大小减少到 12KB
  • 区块索引公共物品 TrueBlocks v1 发布和给以太坊基金会的Grant报告
  • tx2uml v1.1.23：添加选项方便隐藏参数名称/值；v1.1.24支持 Base 链
  • wagmi mipd：用于 ERC6963 （多注入 Provider 发现）的 TypeScript 实用程序
  • CTF：
    • Curta CTF：devtooligan 的 Elohim 解决方案
    • Paradigm CTF 10 月 28-30 日
  • Cryogen：实现更小的数据集和更快的查询

安全

  • Balancer v2 池漏洞曝光，受影响的 LP 赎回

生态

  • Danny Ryan反驳了Lido 的说法：即随着 Lido 持股比例接近 33%，双重治理可以减轻以太坊的风险
  • 黑客马拉松项目：ETHToronto、ETHWomen和以太坊阿根廷
  • Etherscan 添加 搜索栏快捷方式：按 “/” 键快捷进入搜索栏

应用层

  • 隐私和扩展探索Bandada v1：创建/管理隐私保护匿名组
  • ScopeLift：利用 Layer 2 进行 DAO 投票，使用灵活投票、概念证明
  • Base 应用：
    • Friend Tech增加了与 Key 持有者的照片共享功能
    • StoryBase：按角色付费协作写作，以 NFT 形式铸造的片段
    • 自行车比赛模仿：在 10 天的比赛中铸造/窃取 NYC Citi Bikes 的 NFT
