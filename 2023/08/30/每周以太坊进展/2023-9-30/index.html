2023-9-30

发表于 2023-08-30 更新于 2023-10-27 分类于 每周以太坊进展 阅读次数：
本文字数： 2.9k 阅读时长 ≈ 3 分钟

Dencun（Cancun + Deneb）升级

  • 最新的所有核心开发人员 - 执行 (ACDE)视频会议。Tim Beiko的总结。Tim Beiko和Christine Kim的记录：
    • Devnet-9 使用 EIP4788 主网地址和手动部署，并计划测试 MEV-Boost；KZG 仪式可信设置预计下周进行
    • EIP7514（限制质押增长）将在短暂的 Devnet-10 上进行测试，使用大量验证者
    • 从 3 个审计中修复了 EIP4788（信标根）的问题：0 时间戳和环形缓冲区大小
    • 对提案EIP7503（通过重新铸造烧毁的 ETH 实现链上隐私）进行讨论
  • Devnet-9 已上线

中心化观察：威胁你的 ETH 价值

  • 🚨 Lido 达到 32.23% 非常接近危险的 33.3%阈值

  • 客户端多样性（通过clientdiversity.org）：

    • 执行层：Geth 占据约 80%的超级多数派，可能导致链分叉
    • 共识层：Prysm 占据 45%，任何客户端的错误超过 33.3%可能导致链活力丧失

  • 需要更多的地理多样性，包括验证者和节点，特别是在美国/加拿大/欧盟之外

Layer1

  • 91%的区块通过仅有四个实体控制的 MEV-Boost 中继进行
  • 审查仪表盘：展示验证者、中继和构建者的 OFAC 合规性
  • 内存池垃圾箱（Parquet 和 CSV 中的交易）：添加包含状态和包含延迟

Holešky 测试网的客户端发布

  • 共识层：
    • Lighthouse v4.5.0：实验性 QUIC 支持
    • Lodestar v1.11.3
    • Nimbus v23.9.1
    • Prysm v4.1.0-alpha.1
    • Teku v23.9.1：Holešky 测试网在 Linux 上需要 GLIBC v2.34
  • 执行层：
    • Besu v23.7.3
    • Erigon v2.49.3：Holešky；v2.50.0：实现 Cancun EIPs
    • Geth v1.13.2：修复路径数据库存储模式的边缘情况
    • Nethermind v1.21.0：可执行文件重命名，限制 RPC 调用，Trie 恢复和同步时间减少
    • Reth v0.1.0-alpha.9：实现 Cancun EIPs，重新设计了 revm 状态处理 API 和状态修剪改进；v0.1.0-alpha.10：Holešky支持

研究

  • 为 danksharding 做准备：主网上大块传播时间的数据可视化
  • 中继构建的包含列表：提供 MEV-Boost 中的抗审查方案
  • 用于多资源的动态费用优化设计框架

Layer2

  • OP Stack：社交去中心化和欺诈证明虚拟机和争议博弈
  • Roll-op：用于配置和部署 OP Stack 网络的脚本

EIPs/标准

  • EIPs
    • EIP7525: 不活跃账户的过期状态
  • ERCs（应用层）：
    • ERC7524: 钱包中的 PLUME 签名
    • ERC7526: Onchain NFT 版税实施

开发者相关

  • Holešky 测试网

    用于质押、基础设施和协议测试

    • 浏览器：Etherscan、beaconcha.in 和 Otterscan
    • 分发给合约部署者的 1000 holETH（主网和 Goerli 测试网）
    • 水龙头：pk910 PoW 水龙头 和 Etherscan 的 Blockscan 聊天
    • 替代已弃用的 Goerli
    • 用于应用层测试的 Sepolia 测试网

  • Remix v0.36.0: 使用 ChatGPT 解释函数、生成文档和解释编译器错误，增加更多快速修复和对 Ephemery 测试网的支持

  • Hardhat v2.17.4: 支持 debug_traceCall

  • Branching Tree Technique examples: 用于编写结构化 Solidity 测试的规范框架

  • Solmate（合约库）添加了 ERC6909（最小多代币接口）的实现

  • SSTORE gas：

    • OpenZeppelin ReentrancyGuard 避免重置为零 以最大化 gas 退款
    • EVM 使用 SSTORE 的当前值而非中间值 以降低 gas 成本

  • 借贷协议架构比较: MakerDAO、Compound、Aave、Euler 和 Yield

  • Huffathon 挑战：1，2，3和4

  • wagmi v2 alpha：TanStack Query 支持，连接多个连接器，断开连接时切换链和 EIP6963 支持

  • ERC4337（账户抽象）：

    • Permissionless.js：与 bundler （打包者）和 paymaster 进行交互的 TypeScript 库，构建在 viem 上
    • Alchemy的Rundler：用 Rust 编写的 ERC4337 bundler （打包者）

  • Solidity Next.js starter v2：使用Hardhat，viem，wagmi，Tailwind CSS 和 RainbowKit

  • Ether binder beta：用于与以太坊交互的 PHP 实用工具，使用 PHP 8.2

  • Curta CTF Philosopher’s Stone writeup

  • 使用 Cryo 和 Polars 进行数据分析的指南

  • Banteg：创建 ERC20 转账的热力图

  • RareSkills：Circom 教程

安全

  • SEAL 911 成员从 Dice9win 中的漏洞中挽回了 20 万美元
  • GMX（Arbitrum 上的 DEX）披露了价格扭曲漏洞，并支付了 100 万美元的赏金来修复

生态

  • Dankrad：以太坊的价值需要社区保护，它们不会自动受到经济激励的保证
  • EF 数据收集赠款，申请截止日期为 10 月 23 日，心愿单包括XATU（网络监控），测试网和数据收集项目
  • ERC4337（账户抽象）采用分析报告：687,000 个钱包，2,000,000 个用户操作，大多数钱包使用不超过 5 次
  • ETHGlobal 纽约黑客马拉松决赛列表

企业应用

  • 国际清算银行 Mariana 项目：在 Sepolia 测试网上使用 AMM 进行跨境大规模兑换 CBDC 的概念验证
  • 布宜诺斯艾利斯数字身份：QuarkID 钱包可用于申领出生证明和结婚证书，使用 zkSync Era

链上统计

  • Gas 费用（通过ultrasound.money）：5.5 至 74.1 gwei，平均为 10.7 gwei

    • 目前通缩 gas 为 20.9 gwei
    • 本周净新发行量为 7.9k ETH

  • ETHUSD：1,569–1,569–1,684，当前为 $1668

  • ETHBTC：当前为 0.062（Flippening 在约 0.16）

应用层的值得注意的事项

  • Arbitrum Odyssey 重新启动，为用户提供为期 7 周的活动，试用应用并收集徽章
  • Pudgy Toys：Pudgy Penguin 的实体玩具，将在沃尔玛销售，许可费将与 NFT 持有者共享，购买者可以在 zkSync 上申领一个永久的 Pudgy
  • Nouns DAO 燃烧提案：激励使用财库资金，否则多余的 ETH 可以被燃烧
  • Paris Hilton 推广 MareBear NFT：Tamagotchis 宠物

其他

  • VanEck 宣布即将推出的 ETH 期货 ETF（EFUT）
  • Coinbase 获得百慕大货币管理局批准，提供永续期货交易，非美国用户可用
  • 纽约和波士顿联邦储备银行：稳定币的安全飞行动态 类似于货币市场基金
  • Chase UK 将拒绝加密货币相关支付
  • 交易所黑客事件：Mixin Network 损失 2 亿美元，HTX（原名火币）损失 800 万美元
  • Sigmabus：在通用 SNARK 电路中高效计算椭圆曲线群操作
  • Naysayer 证明：乐观地接受证明，并在必要时进行检查
