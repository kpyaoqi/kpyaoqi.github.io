2023-9-16

发表于 2023-08-21 更新于 2023-10-27 分类于 每周以太坊进展 阅读次数：
本文字数： 1.8k 阅读时长 ≈ 2 分钟

Dencun（Cancun + Deneb）升级

  • 最新的所有核心开发人员 - 执行 (ACDE)视频会议。Tim Beiko的总结。Tim Beiko和Christine Kim的记录：

    • 在 kurtosis 上继续对 devnet-8 和 MEV-Boost 工作流进行测试
    • 在 Dencun 中添加了最大 epoch 流失限制（EIP7514），每个 epoch 的最大新质押者将被设置为 8；
      Dankrad：EIP7514 给时间处理 Lido 攻击的经济/技术影响
    • 在 Dencun 中添加了用于 L2 费用计算的 BLOBBASEFEE 操作码（EIP7516）
    • EIP4788（EVM 中的信标块根）正在更新审计提出的调整
    • Devnet-9 将包括 EIP7514、EIP7516 和 EIP4788 的更改，ACDC 将确定启动日期
    • Reth 介绍：用 Rust 实现的执行层客户端，希望加入 devnet-9

中心化观察：威胁你 ETH 的价值

  • Lido 达到 32.3%，虽然有所减少，但仍非常接近33%阈值

  • 客户端多样性（通过clientdiversity.org）：

    • 执行层：Geth 占据绝对多数，可能导致链分叉
    • 共识层：Prysm 超过 33%，可能导致活跃性丧失漏洞

  • 需要更多的质押者和节点地理多样性，尤其是在美国/加拿大/欧盟以外的地区

Layer 1

  • epoch 流失限制从 11 增加到 12（Dencun 中的 EIP7514 将存款设置为 8）
  • MEV-build-rs alpha：用 Rust 实现的 MEV-Boost 构建器，扩展了 Reth（执行层客户端）

给质押者

  • Lodestar 激励计划 结果，86 个验证者瓜分 25,000 美元
  • Geth 节点 在 NanoPC-T6（价值 129 美元的板）上运行，13 小时同步完成
  • Secure-signer（alpha 版）：硬件隔离的远程签名
  • Tor Push：验证者隐私的概念验证，通过 Tor 将消息推送到 gossipsub

客户端版本

  • 共识层：
    • Nimbus v23.9.0：Holešky 测试网支持和简化的远程签名配置
  • 执行层：
    • Geth v1.13.0：用基于路径的存储重新实现修剪（需要重新同步）

研究

  • 简化的快速最终性：每个 slot 只有一个提案和投票阶段的强确认
  • 两层质押：本地流动质押设计

EIP

  • EIPs
    • EIP7516：BLOBBASEFEE 操作码
  • ERCs（应用层）：
    • ERC7515：元数据中的 NFT 替代文本
    • ERC7517：AI/ML 数据挖掘的内容许可
    • ERC7518：全球合规的 STO 和可互操作的资产生命周期（ERC1155 扩展）

开发者资料

  • Holešky 测试网未能启动，因为配置错误，预计两周后重新启动
  • Hardhat v2.17.3：默认evmVersion修改为paris以适用于 solc v0.8.20 以上的版本（适用不支持上海升级的 L2）
  • 如果直接从汇编块返回，就会跳过修饰符中的不变式检查
  • 最小代理概述：不可变参数的 ERC1167 变体和克隆
  • ERC4626 一致性：20% 的资金库不符合规范
  • MinimalAccount（ERC4337）：用 Huff 实现的 gas 优化，用于基准测试
  • ERC 部署：Dune 仪表板显示常见 ERC 的部署数量
  • Huff-stack-generator：生成堆栈注释
  • Geas（优秀的以太坊汇编）：EVM 的宏汇编器
  • Slitherin（自定义 Slither 检测器）v0.3：添加任意调用检测器
  • Vyper 编译器审计竞赛
  • CTF：
    • ONLYPWNER：CTF 平台，有9个挑战
    • Curta CTF 谋杀之谜和解决方案
    • Secureum RACE #21：8 个问题的 Solidity 测验和答案
  • Paradigm 的Rust x Ethereum 幻灯片和视频：Reth、Foundry、Alloy、Ruint 和 wagmi/viem & Rivet
  • ZK Face ID 钱包（ERC4337）演示，用 WebAuthn 和 Halo2 证明

安全

  • Balancer 两个 Linear Pools 漏洞的事后分析：第一个漏洞悬赏了 100 万美元，第二个漏洞在主网和 Optimism 被利用，造成 120 万美元损失
  • 寻找漏洞：发现了一个 Premia 漏洞，300 万美元存在风险，已悬赏并修复

应用层

  • MetaMask Snaps beta：用于交易安全、聊天和非 EVM 网络的钱包插件
  • Uniswap UI：自动检测转账代币的费用
  • Gaslite：空投 ETH、ERC20 和 ERC721 代币，包括 ETH 给 Friend Tech 的密钥持有者
  • Optimism：直接分发未领取的 Airdrop 1 的 OP 代币
