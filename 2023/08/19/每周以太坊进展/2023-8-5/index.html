2023-8-5

发表于 2023-08-19 更新于 2023-10-27 分类于 每周以太坊进展 阅读次数：
本文字数： 1.8k 阅读时长 ≈ 2 分钟

Dencun (Cancun + Deneb) 升级

  • 最新的所有核心开发者 - 共识（ACDC）视频会议。克里斯汀·金 (Christine Kim)的注释：

    • EIP4788（在 EVM 中记录信标块根）：同意使用合约而不是预编译，需要审计/形式化验证，devnet 8 将等待实现它

    • EF devops 推出了 210 万个验证器测试网，为 Holešky 测试网做准备

    • EIP5806（委托交易）和 EIP7377（迁移交易）的介绍以及关于帐户抽象的更广泛讨论

    • Verkle Trie 更新: 性能改进、通过覆盖方法进行迁移以及还有一些剩余问题。

  • 共识规范 v1.4.0-beta.1: 数据 gas 更名为 blob gas 以及包含附加测试

  • EIP4844:

    • Terence: EIP4844 经济学
    • KZG 仪式 有133k捐款, 贡献大厅空了; 特别贡献清单 (从卫星到猫)
    • GossipSub 优化 (IDontWant & 限制 flood 发布) 应会减少 Nimbus CL 客户端带宽消耗

Layer 1

  • Reorg.pics: 主网重组的仪表板

  • Goerli 测试网上的长访问的 Epoch 发生了重组

    , 不太可能在主网上发生

    • Ben Edgington在《升级以太坊》一书中的解释

  • SSZ++: 快速 SSZ 库，27 毫秒反序列化和 23 毫秒完整信标状态哈希，无需缓存

  • evmone 中的 EVMMAX API 原型 : 允许实现大多数预编译

给质押者

  • Wenmerge 非过滤 MEV-Boost 中继

客户端更新

  • 共识层：
    • Lodestar v1.10.0: 性能改进、NodeJS v20、IPv6 支持、带宽优化和 DVT 用户的 builderOnly 选项
    • Teku v23.8.0: 状态缓存改进和最低 Java 版本更新至 Java 17

研究

  • Vitalik: 数据可用性采样中数据可恢复性的细微差别
  • 协议强制提议者承诺 (PEPC) 常见问题解答
  • 中继后 ePBS & 提案最顶区块(ToB)支付
  • Potuz: ePBS 中的三个二分法

Layer 2

  • Offchain Labs BOLD (有限流动性延迟): 提出了用于 Arbitrum 链无许可验证的争议协议，消除了延迟攻击向量
  • Popcorn: zkEVM 互相批评归因 和营销主张

EIPs/标准

  • EIPs:
    • EIP7441: 将区块提议者选举升级为 Whisk
  • ERCs (应用层):
    • ERC7425: 代币化储备
    • ERC7432: NFT 角色
    • ERC7439: 防止票务兜售（ERC721扩展）
    • ERC7444: 时间锁到期

给开发者

  • Hardhat v2.17.1: Solidity v0.8.21 支持、纯函数中的 console.log 和改进代码扁平化

  • Foundry:

    • 提示: 使用 lcov 生成本地覆盖率报告 using lcov

  • Dan Finlay’ Foundry 的 MATT 拍卖 (最大化 NFT 收入)， 使用 ERC20 授权来出价

  • web3.py WebSocketProviderV2: 重写WebsocketProvider, beta版

  • CryptoZombies Optimism NFT 课程: 创建、测试和部署

  • DeCipher: 使用区块浏览器 URL 的合约文档生成器

  • Secureum RACE #20: 8 个问题 Solidity 测验和答案

  • 安全

    • Vyper 在 v0.2.15/16 和 v0.3.0 中重入锁故障,Curve 价值 7000 万美元的漏洞影响了 Alchemix, JPEG’d & Metronome, 返回了一些Alchemix&JPEG’d的资金

      • 潜在 脆弱的 Vyper 合约

    • Base 上的LeetSwap (Solidly 分叉) 被利用, 池通过内部函数被作为公开函数，而被操纵

    • 2000 万美元发送到 垃圾投毒地址, USDT 随后被冻结

    • Whitehacks 套件: 使用 Flashbots 在单个交易中执行操作的 Foundry 模板

    • 使用注释的 Solidity 审计报告生成器 (VSCode 插件)

生态相关

  • 以太坊主网推出 八周年
  • Ethereum.org 翻译马拉松: 翻译网站即可赢取奖品
  • Tx 聊天对话: 类似聊天的界面，用于浏览链上消息的地址
  • Golem Octant Epoch Zero: 100 万美元分配给 10 个公共产品项目
  • Curve 联合创始人 Michael Egorov 在 Curve 漏洞利用后去杠杆化
  • 用于预防垃圾攻击同时保持匿名的 Rate-limiting nullifier (RLN)， 为可信仪式做贡献
  • Gitcoin Grants 18 申请开放
