2023-7-8

发表于 2023-08-02 更新于 2023-10-27 分类于 每周以太坊进展 阅读次数：
本文字数： 1.5k 阅读时长 ≈ 1 分钟

Dencun (Cancun + Deneb) 升级

  • 最新的所有核心开发者会议 – 执行（ACDE）视频会议。来自Christine Kim的说明：
  • EIP6404/6466 SSZ 过渡的影响对使用RLP的现有合约的影响：EIP6404 的影响较小，而EIP6466 由于 3个项目受影响，依旧是可以管理 【提醒：SSZ EIPs已经从 Dencun 升级中删除】
  • Dencun更新：devnet 问题跟踪，计划在devnet 7上对EIP4844进行压力测试，同时并行为 devnet 8 开发全部的 Dencun EIPs。
  • 讨论 EL 的引擎 API 标志，该标志用于建议 CL 使用本地块构建
  • EIP4844：
    • 为 KZG 仪式做贡献的最后机会，7月23日截止，使用至少有8个交易的账户

客户端发布

  • 共识层：
    • Lighthouse v4.3.0：减少带宽，并对slasher进行了修改：默认启用广播，后端恢复为LMDB。
  • 执行层：
    • Reth v0.1.0-alpha.2：引擎API的稳定性得到改善，修复了内存不足的问题，修复了RPC 边缘情况，包括tracing和增加了新的指标。

给质押者

  • MEV-Boost dashboard 增加了乐观中继的图表，显示了增加的胜率

研究

  • 纳入PBS的建议：payload-timeliness committee (PTC)，证明委员会的子集对构建者是否及时发布其执行 payload 进行投票。
  • 交易费用机制设计，证明了 MEV 使得用户体验不可能简单，验证者也不可能被激励去做正确的行为
  • 分布式区块构建：使用MPC进行保护隐私的knapsack拍卖

Layer2

  • Patrick McCorry: 跨链桥的概述

EIP 标准

  • EIPs：
    • EIP7266：移除blake2f (0x09)预编译
  • ERCs（应用层）：
    • ERC7265：断路器，用于当超过阈值时暂停 Token 留出。
    • ERC7272：以太坊访问代币（授权来自链外服务的函数调用）
    • ERC7280：NFT元数据扩展，如JSON-LD
    • ERC7281：主权桥接代币(xERC20)
    • ERC7291：目的绑定货币（ERC1155 扩展）

开发者资料

  • 使用 Diffyscan 对 GitHub 合约和 Etherscan 验证的代码进行比较
  • Scaffold-ETH 2 更新至Wagmi v1 & viem
  • Svelte Wagmi：Svelte stores and functions 使用wagmi进行连接和签名
  • Vitest-solidity-coverage：用于 Vitest 的solidity-coverage的Hardhat插件。
  • Ape v0.6.12（合约开发框架）：增加了EIP5202 蓝图支持，覆盖和安装与编译的依赖性。
  • Huff 代码风格指南
  • RareSkills：合约安全工具概述
  • CTF（挑战）：
    • Curta CTF: 乒乓球
    • Secureum RACE #19: 8个问题的Solidity测验和答案
  • 创建自定义Uniswap v4 Hook 的指南
  • Reth-indexer：从Reth（EL客户端）数据库中读取，索引到Postgres并生成API。
  • Circomkit：简单接口和测试工具， 用于开发Circom电路。

安全

  • Azuki DAO（不满的Azuki持有者）由于缺失的签名重放检查漏洞导致 35 ETH 被利用。
  • Tincho: 问题驱动的方法来审查代码
  • Multichain (即Anyswap)跨链桥1.26亿美元的漏洞
  • Poly Network 跨链桥400万美元的漏洞，4个私钥中的3个明显被滥用

生态系统

  • Etherscan 添加 ENS到搜索栏自动查询。

可关注的应用层动态

  • Zora network (OP Stack L2) 由OpenSea支持。
  • SUBMINT (ENS子名铸币)在主网上线，设置定价和代币门铸币
  • 本周的Gas guzzler ：VMPX ERC20代币需要工作证明才能铸币
