2023-8-19

发表于 2023-08-07 更新于 2023-10-27 分类于 每周以太坊进展 阅读次数：
本文字数： 2.2k 阅读时长 ≈ 2 分钟

Dencun (Cancun + Deneb)升级

  • 最新的所有核心开发人员 - 执行 (ACDE)视频会议。Tim Beiko的总结。Tim Beiko和Christine Kim的记录：

    • Devnet-8 启动，所有 Dencun EIP、Besu 和 Erigon EL 客户端仍待加入

  • EIP4788（EVM信标块根）合约将以单独的交易部署

    • Holešky 测试网将支持 16亿 ETH 供应量

  • ERC 从 EIP 库中分离出来，EIP 编辑团队仍是同一个组织

    • 提出 EIP7212（用于 secp256r1 椭圆曲线中签名验证的预编译）

  • KZG 仪式已有 138k 贡献者，将于 8 月 23 日停止接收贡献

Layer 1

  • 集成 builder-searcher 比率提升 ，动画图表显示自合并以来各类构建者每月构造区块份额的变化过程
  • Dapplion：Whisk（SSLE）导致错过初始 slot，提案修复边缘情况

给质押者

  • 出现内存不足错误的 Lodestar v1.10 节点，建议将 NodeJS 版本升级到 nodeJS v20
  • Obol DVT 性能测试：与传统验证者差异小于 1%

客户端版本

  • 执行层：
    • Geth v1.12.2：修复了一些社区报告的回归测试发现的问题

研究

  • 提案包含列表设计，分为提议者签名摘要和未签名的交易列表，解决免费数据可用性问题

Layer 2

  • Arbitrum One 批量发布者在 Gas 价格飙升期间短暂停止工作，用户仍然可以进行交易
  • Rollup 经济学 v2：运作一个有盈余、福利社、联盟和layer 3

EIP

  • EIP/ERC 编号从 7500开始改为连续编号， 而不是使用 PR 编号
  • EIP：
    • EIP7495：SSZ PartialContainer： 引入一个新的 Simple Serialize ( SSZ )类型来表示PartialContainer [ T , N ]值。 PartialContainer [T, N] 是一个 SSZ Container，即使单个字段成为可选字段或将来引入新字段，也具有稳定的序列化和默克尔化。
  • ERC（应用层）：
    • ERC7484：智能合约帐户的注册表适配器
    • ERC7492：ERC20Spendable（ERC20扩展）
    • ERC7500：NFT动态特征
    • ERC7501：可赎回 NFT

开发者资料

  • Remix v0.35.0：添加了许可证、可见性和可变性的快速修复功能
  • Foundry:
    • SKS：代替子模块（Git Submodules），用于安装依赖项的包管理器脚本
    • 提示：在合约导入路径中使用 src以兼容 Hardhat
  • OpenZeppelin Contracts 低严重性问题：ERC2771Context与自定义转发器一起使用可能导致_msgSender返回address(0)，已在 v4.9.3 中修复
  • RareSkills：Solidity 风格指南
  • 具有 immutable 参数的 Vyper 克隆：与 CWIA 代理工厂兼容的示例合约
  • 用 Vyper 实现的 ERC6909（最小多代币接口）
  • Sol2uml diff可以将 Etherscan 验证的合约与本地文件进行对比
  • WhatsABI v0.7（从字节码猜测 ABI）：支持代理合约
  • heimdall-rs v0.5.0添加合约快照模块
  • Nethereum v4.16 (.NET)：添加数据服务包以便与 Etherscan/Sourcify/4Bytes 交互、使用 RPC 批处理的多重查询， ABI 解码和 ABI 存储的额外扩展
  • 钱包测试框架测试：当前需要 window.ethereum（不包括移动钱包）和自定义网络支持
  • Cryo v0.2（提取区块链数据）：添加 python 适配器，合约和 native_transfers 数据集、交易选择器、命令链和摘要报告
  • Dune SQL 和以太坊数据分析高级指南
  • ERC5564（隐形地址）bn254：使用 arkworks-rs 和 RLN 为属于其他成员集的用户生成可衍生的承诺并将其添加到其他成员中

安全

  • ExactlyProtocol 通过未经验证的输入漏洞被利用 700 万美元
  • Zunami 协议通过捐赠操纵价格200 万美元漏洞被利用
  • RocketSwap 部署者私钥泄露87 万美元被利用，
  • 审计工具箱：带有 Foundry、Halmos、Heimdall、Slither、Echidna 和 Mythril 等的 docker
  • 审计向导：使用合约地址或 GitHub，使用 Slither 扫描，查看 AST 并生成图表，测试版
  • Trail of Bits：Rekt Test 12 个用于评估安全性的是/否问题（以 Joel Test 为模型）

生态

  • Gitcoin Grants 18已开放

    欢迎大家为 登链社区 和 Decert.me 捐赠

  • EF 第二季度受资助者将分享 900 万美元资金

  • ETHGlobal Superhack（OP Stack）决赛入围者

  • ETH慕尼黑黑客马拉松获奖者

应用层

  • OpenSea 创建者版税变为可选，包括使用市场过滤器的 L2 和 NFT
  • CoW Swap 时间加权平均价格 (TWAP) 订单
  • Connext 治理代币，使用 xERC20 跨链空投
  • Anish 的FriendMEX，Friend Tech 的专业风格界面，无需邀请码
  • Brian Armstrong：Coinbase 需要改进 NFT、Dapp 和 L2 的应用程序用户体验

其他

  • Vitalik：发表 X（Twitter）社区笔记，具有“加密价值”的机制设计
  • Patrick McCorry：发表 非洲的加密货币
