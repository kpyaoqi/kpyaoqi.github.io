2023-8-12

发表于 2023-08-04 更新于 2023-10-27 分类于 每周以太坊进展 阅读次数：
本文字数： 2.1k 阅读时长 ≈ 2 分钟

Dencun (Cancun + Deneb) 升级

  • 最新的所有核心开发人员 - 执行 (ACDE)视频会议。Tim Beiko的总结。Tim Beiko和Christine Kim的记录：

    • Devnet 8 计划于下周初启动
    • 共识层分叉选择过滤逻辑的更改将与 Deneb 版本捆绑在一起

  • EIP4788（EVM 信标块根）合约草案，欢迎来 gas 优化

  • 最新 Dencun 测试视频会议（之前的 EIP4844 会议）

  • KZG 仪式已有 136k 贡献者，将于 8 月 23 日停止接收贡献

Layer 1

  • EthereumJS v7：支持 ESM ，删除缓冲区，降低依赖性，减小构建大小，现在默认支持上海升级和 Cancun 升级
  • EF devops 启动了 3 倍主网规模（210 万验证者）的大型测试网来测试目前的客户端状态能处理多大的网络，三次尝试仍有问题，改为2倍主网规模(140万验证者)后出块速度接近主网，Holešky 测试网会将140万验证者的规模视为稳定起点
  • Kurtosis eth2-package，支持云中多节点执行层/共识层测试网和任何客户端组合（包括 Reth），与 Grafana 和 Prometheus 一起开箱即用
  • Verkle Tries：
    • Verkle 原像生成和分发策略
    • Verkle Trie 迁移选项：只读与可写 Merkle 树
  • Flashbots 构建器故障，在恢复捆绑交易的过程中处理不当

给质押者

  • 信标链的churn 限制已增加到每个 epoch 11 个验证者
  • 目前运行以太坊节点最便宜的配置-NanoPC-T6，约 300 美元，可运行完整节点
  • bloXroute Ethical 中继正在逐步废止（过滤广义抢跑交易和夹子交易）

客户端版本

  • 执行层：
    • Geth v1.12.1：维护版本，包括 Dencun 的开发
    • Reth v0.1.0-alpha.6：跟进修复 alpha.5 中处理重组的问题

研究

  • 第二个 slot 重组：slot 指数影响重组的可能性
  • Builder 行为概况(BBP)：涵盖 MEV-Boost 拍卖期间构建区块和投标时的功能和策略的指标
  • 强制协议提案者承诺(PEPC) 介绍
  • 抗量子弱可验证延迟函数，一种基于纯代数构造的新的抗量子弱可验证延迟函数。它的延迟取决于计算椭圆曲线之间的高度同源性，而其验证则依赖于两条椭圆曲线乘积的同源性计算。

Layer 2

  • Coinbase Base（OP Stack rollup）向公众开放（链上夏季促销活动）

EIP

  • EIP：
    • EIP7480：EOF – 数据指令

开发者资料

  • Solmate v6.2.0：修复 SignedWadMath:wadMul 中的静默溢出问题；这个问题是用 Trail of Bits Medusa发现的
  • Iter：用 Solidity 实现的惰性评估迭代器
  • Cove：合约验证，alpha 版， 为 Solidity Foundry 项目准备，使用部署 tx 哈希进行合约验证
  • EVM Diff 更新
  • Bulloak：基于分支树技术的 Solidity 测试生成器
  • 深入研究 Vyper 编译器
  • Fe v0.24.0：增加支持用户定义库
  • Halmos v0.1.2（符号测试）：添加支持 DELEGATECALL、CALLCODE、CREATE2、结构类型作为测试参数和库链接
  • Agent Buttercup：面向 DeFi 研究人员/工程师的基于代理的模拟引擎，使用 revm
  • yAcademy zk 审计奖学金模块：zk、Rate-limiting Nullifier 和 Spartan-ECDSA
  • CTF：
    • Flashbots MEV-Share CTF 解决方案
    • Curta CTF 公牛比利解决方案
  • p0tion v1：Groth16 zk 应用程序开发工具包
  • Noble curvetyposquat：恶意 npm包将私钥发送到远程服务器

安全

  • Steadefi 在 Arbitrum 和 alt-L1 上通过被入侵的协议部署者钱包被盗 110 万美元
  • UniswapX 订单漏洞披露，已修复并支付 20 万美元赏金
  • Vyper v0.2.15/16 和 v0.3.0 中不可重入锁漏洞的事后分析
  • SEAL 911：紧急情况下通过 Telegram 机器人联系安全社区

生态

  • Danny 回应那些试图让中心化变化变常规的人：Lido 超过 33% 的 ETH 质押对以太坊构成威胁
  • EF 下一个十亿奖学金：第 3 组的五个人物故事
  • Etherscan相似合约搜索更新，可以跨 Etherscan 链搜索

应用层

  • Rune 的Maker 终极之战和SubDAO 挖矿概述
  • Maker增强型 DSR（Dai 储蓄率）从 8% 开始，降至 5.8%
  • Nexus Mutual通过 Retail Mutual 为英国小型零售商提供覆盖
  • 安全社交登录：在 iOS 上使用 Google 或 Apple ID 注册，实验性
  • MultiSafe：将 Safe 克隆或部署到具有相同地址的多个链
  • Y00ts NFT 在之前从 Solana 迁移到 Polygon 后迁移到以太坊
  • DAO 关闭并分发 Arbitrum 空投：Saddle（Curve fork）和Hundred Finance
  • Base 应用：
    • BasePaint：共享像素画布，每24小时开放版NFT
    • Words3：价格波动的字母拼字游戏，游戏本周末结束
    • Friend Tech：在移动设备上通过 DM、PWA 购买人员股份，仅限受邀者
