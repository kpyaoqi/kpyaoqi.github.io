2023-6-17

发表于 2023-07-12 更新于 2023-10-27 分类于 每周以太坊进展 阅读次数：
本文字数： 1.8k 阅读时长 ≈ 2 分钟

Dencun (Cancun + Deneb) 升级

  • 最新的所有核心开发人员 - 执行 (ACDE)视频会议。Christine Kim的记录：

    • Deneb 范围：EIP7044（签名退出永久有效）和 EIP4788（EVM 中的信标链区块根）；考虑包含：EIP7045（扩大证明slot包含范围）和 EIP6988（防止被罚没的验证者被选为区块提议者）

  • 关于主网区块重组增加的讨论，提议更改区块生产、证明和聚合的sub-slot时间

    • 关于取消每个验证者最多质押 32 ETH 的讨论

  • EIP4844：

    • 最新的 EIP4844 实施者视频会议。Terence 的记录
    • Dankrad对每个区块可以支持的 blob 大小进行测试
    • EIP4844 devnet 6 测试网发布
    • KZG 仪式有超过 11 万贡献者，没有人排队

  • Consensus-specs v1.4.0-alpha.3：补丁将 BLOB_SIDECAR_SUBNET_COUNT 增加到 6

Layer 1

  • Erigon 存档节点在小于 500 美元的 ARM 板上运行
  • Dedaub：EIP6404/6466 研究用 RLP 的合约向 SSZ 过渡的影响[提醒：SSZ EIP 已从 Dencun 升级中移除]
  • Beacon API 检查器：使用 Rust 跨客户端检查 API 响应的 CLI

客户端版本

  • 共识层：
    • Prysm v4.0.6：修复错误，缩短边缘情况区块提议时间
    • Teku v23.6.0：奖励 beacon-api 端点，更新解决5月的最终性问题
  • 执行层：
    • Erigon v2.45.2：分阶段同步修复

给质押者

  • Payload 的区块分析仪表板，测试版

研究

  • 关于 5 月的最终性问题事件的数据可视化

Layer 2

  • Optimistic Erigon 客户端支持 Optimism 主网
  • Arbitrum定序器批量发布问题分析
  • Layer 2 MEV和去中心化排序
  • Frax Finance 计划 Fraxchain rollup 用它的 LSD 作为 gas 费代币

EIP

  • ERC：
    • ERC7169 : 分批排序
    • ERC7182：接口命名约定

开发者资料

  • Foundry：

    • Pre-v1 更新：fuzz-runs CLI 标志、0 basefee、Optimism 上的批量交易和 readCallers 作弊码
    • Fuzzy DeFi：Uniswap v2、Olympus DAO 和 Compound v2 分叉的代码属性
    • Forge-safe：用 Forge 脚本构建安全的多签批量交易

  • OpenZeppelin 合约v4.9.2：MerkleProof 库补丁，修复processMultiProof和processMultiProofCalldata中的bug，即如果树在深度1处包含一个值为0的节点，则允许证明任意叶子。

  • Uniswapv4：使用钩子自定义流动性池，在 Dencun 升级中使用 ERC1153（瞬态存储）进行闪存记账，Uniswap v4 依旧正在开发中，源码使用：Business Source License ，4 年内限制其他产品使用。

    • V4-template：用于创建 Uniswap v4 钩子的模板
    • Huff 钩子：Huff 中的 Uniswap v4 钩子库，正在进行中

  • Bytecode.zip：从浏览器部署压缩和包装合约

  • Titanoboa（Vyper 解释器）与jupyter notebooks 集成：通过浏览器签名

  • Ethers.js ENS 多币 provider 插件 扩展

  • ABIType：在运行时和类型级别添加从 JSON ABI 到人类可读 ABI 的转换

  • RainbowKit v1.0.2：支持 WalletConnect v2 支持

  • 4byte collider：用于查找有碰撞的4字节选择符的函数签名

  • 钱包测试框架：切换到 viem 并添加了更多测试

  • 用于合约钱包的基于密钥的帐户抽象签名

安全

  • Sturdy Finance，80 万美元漏洞被利用，通过只读重入进行价格操纵
  • 签名延展性：展示了如何执行使用紧凑签名的签名延展性攻击。

应用层

  • Maker Dai 储蓄率 (DSR)于 6 月 19 日升至 3.49%
  • Uniswap v4 愿景：更通用的 v3 ，允许通过自定义池的挂钩进行不同的权衡
  • Uniswap 基金会跨链桥评估报告
  • EigenLayer 第 1 阶段在主网上线，达到启动限制；restaking仪表板
  • Sound（音乐 NFT）在 Optimism 上线
  • Goose NFT 以 540 万美元的价格从 3AC 系列中售出
  • NFT 基础设施项目最近艰难前行：JPG暂停和NFF（支持）逐渐减少
