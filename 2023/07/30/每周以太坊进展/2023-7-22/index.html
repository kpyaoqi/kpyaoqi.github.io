2023-7-22

发表于 2023-07-30 更新于 2023-10-27 分类于 每周以太坊进展 阅读次数：
本文字数： 1.7k 阅读时长 ≈ 2 分钟

Dencun (Cancun + Deneb) 升级

  • 最新的所有核心开发人员 - 执行 (ACDE)视频会议。Tim Beiko的总结。Tim Beiko和Christine Kim的记录：

    • Dencun 测试继续与 devnet 7 和Sepolia 影子分叉一起进行。

  • 实施 Dencun EIP 的客户团队已为 devnet 8 做好准备

    • EIP6780（仅在同一 tx 中自毁）边缘情况澄清，某些 L2 使用了 burn 功能

  • 讨论给执行层有效载荷中添加父信标块根、从执行 API 中添加 getBlockReceipts 和移除 minting ，以及标准化 JWT 存储位置

    • EIP/ERC repo 拆分（EIP7329）：核心开发人员赞成，Greg Colvin（EIP 编辑）仍反对

  • KZG 仪式 122k 贡献者，截止日期为 7 月 23 日。

Prague + Electra 升级

  • EIP7251 FAQ：将允许超过 32 个 ETH 质押
  • 为Prague + Electra 升级版提交 建议名称

Layer1

  • Ben Edgington：升级以太坊书 加新章节：关于 LMD GHOST 和 Casper FFG

客户端发布

  • 共识层：
    • Lodestar v1.9.2：针对出现未知区块父错误的传播区块的热修复，并更改了区块生产等待截止时间
    • Nimbus v23.7.0：优化验证器客户端并提高验证器性能
    • Teku v23.6.2：小错误修复和依赖关系更新
  • 执行层：
    • Nethermind v1.20.1：修复了 v1.20.0 导致的 Prysm 错误
    • Reth v0.1.0-alpha.4：删除了阻止同步的下载器探索、UNIX 上的 jemallocator 默认值，并添加了交易池维护指标

给质押者

  • 质押目录网站：质押参考大全

研究

  • 数据可用性抽样（DAS）电子版：证明现有结构的安全性，提出具有基准（benchmark）和权衡的新结构
  • 阈值加密交易池：使用阈值加密交易池缓解 MEV 的交易问题

layer2

  • Celo 提出向 Rollup 迁移，最初为 OP Stack，使用 Celo 验证器集的去中心化定序器，通过 Eigen 层和 1 个区块最终性实现链外数据可用性
  • zkSync Era 覆盖测试Boojum（基于STARK的证明系统）
  • Mantle（乐观 Rollup ）alpha 已上线
  • Consensys Linea (zk rollup) alpha 已上线
  • L2BEAT: Layer2 的升级过程 [PDF]

EIPs/Standards

  • ERC（应用层）：
    • ERC7336：跨多个注册器的交互
    • ERC7361：具有分布式验证功能的元数据Hash（ERC721 扩展）

给开发者

  • Solidity v0.8.21：
    • 语言特性：放宽从其他合约访问事件和 immutable 变量初始化的限制
    • 解决基于 IR 代码生成的未优化代码的 “堆栈过深 “错误
    • 错误修正：传统代码生成 始终为 .selector 中的表达式生成代码 和 Yul 优化器 FullInliner 评估顺序
    • 网站 改版
  • 创建 质押与奖励合约
  • 使用 Hardhat 或 Foundry 在 Solidity 中重现简单的 MEV 攻击
  • 使用Echidna模糊测试链上合约指南
  • Sothis v0.4.0: 在本地 Anvil/Hardhat 节点上重放历史状态
  • TheGaph：基于 SubStream 的 subgraph
  • Secureum A-MAZE-X CTF 解决方案
  • Celer Brevis：zk 协同处理和数据认证 SDK，alpha版

安全性

  • Conic Finance 因只读重入 $300百万 被利用
  • Immunefi：十大常见漏洞

生态系统

  • EthCC：
    • 各阶段视频： main, Eiffel, Versailles, Bastille, Notre Dame, Louvre & Saint Victor
    • EthCC 2024将在布鲁塞尔举行
  • Etherscan 高级过滤器更新

应用层

  • Aave GHO（超额抵押稳定币）在主网上上线，Aave v3 提供的资产被用作抵押品
  • UniswapX(聚合器)发布：无Gas SWap，有 MEV 保护，交易失败无费用，测试版
  • Ajna（借贷）在主网上线，无治理，无外部价格预言机
  • OpenSea Deals：创建用 NFT 和 WETH 交换 NFT 的出价
  • Mint.fun（NFT 铸币）增加了对 Optimism 和 Zora 上 NFT 的支持
