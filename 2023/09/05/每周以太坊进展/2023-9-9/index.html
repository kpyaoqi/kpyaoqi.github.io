2023-9-9

发表于 2023-09-05 更新于 2023-10-27 分类于 每周以太坊进展 阅读次数：
本文字数： 2.2k 阅读时长 ≈ 2 分钟

Dencun（Cancun + Deneb）升级

  • 最新的所有核心开发人员 - 执行 (ACDE)视频会议。Christine Kim的记录：

    • 在 devnet-8 上继续测试
    • 计划于 9 月 19 日星期二进行 devnet-9
    • EIP4844 每个区块的 blob：大区块分析 报告，目前没有对 3/6（目标值/最大值）做任何更改
    • 提议添加到 Dencun（开放给社区输入）：最大epoch churn 限制作为验证者集大小增长的短期修复，ACDE 上讨论是否包含在 Dencun 中
      ​

中心化观察：威胁你 ETH 的价值

  • 🚨 Lido 达到 32.4%非常接近危险的33%阈值

  • 客户端多样性（通过clientdiversity.org）：

    • 执行层：Geth 超过半数，可能导致链分叉
    • 共识层：Prysm 和 Lighthouse 超过 33%，一个错误可能导致丧失活性(liveness)

  • 需要更多的地理多样性，包括质押者和节点，特别是在美国/加拿大/欧盟之外
    ​

第一层

  • Mempool Dumpster：以 Parquet 和 CSV 格式导出 EL 节点交易池中的交易
  • MEV-Boost：
    • 增强构建者身份：提议用于对抗冒名顶替攻击（Titan 曾有冒名顶替者）
    • Prestwich：提议者和构建者可能串通，限制gas以最小化基础费用（base fee）
  • Whisk devnet：在 Lighthouse + Geth 的分叉上进行概念验证的单一秘密领导者选举（SSLE）
    ​

给质押者

  • 质押大全站点（Staking directory）：添加了关于网络共享和自我限制质押的信息
    ​

Holešky 测试网的客户端发布

  • 共识层：

    • Lighthouse v4.4.1
    • Lodestar v1.11.1
    • Teku v23.9.0

  • 执行层：

    • Besu v23.7.2

    • Erigon v2.49.0

      • Otterscan v2.0.0 alpha：添加了节点内的token索引器

    ​

研究

  • Terence 的EIP4844+（Dencun 升级后扩展 blob 的想法）：擦除代码以增加目标/最大 blob 数为 6/12，并应用数据可用性抽样
  • PeerDAS：重用 P2P 组件的数据可用性抽样方法，以实现无需 DHT 的扩展
  • Barnabe：包含列表游戏，建设者可能会被激励填充区块，提议者可能会试图贿赂/勒索对方

Layer2

  • Base 区块生产延迟为 45 分钟
  • Barry Whitehat 的Plasma Free提案：免费交易，但如果数据不可用，用户需要进行强制交易（基于在线用户的假设）

EIPs/标准

  • EIPs
    • EIP7514：添加最大epoch churn限制
  • ERCs（应用层）：
    • ERC7507：多用户NFT扩展（ERC721 扩展）
    • ERC7508：动态的链上Token属性存储库
    • ERC7509：实体组件系统
    • ERC7510：跨合约分层NFT（ERC721 扩展）
    • ERC7511：带有 PUSH0 的最小代理合约
    • ERC7512：用于审计的链上表示
    • ERC7513：NFT 绑定的模块化合约（ERC1155 扩展）

开发者相关内容

  • Foundry:
    • Forge-std v1.6.1：新的作弊码，用于获取映射的键、文件系统作弊码 和 JSON 解析
    • Catapulta（Foundry 部署自动化）：抽象了 RPC 配置和Etherscan keys，在 alpha 版本期间免费
    • Solpretty：用于漂亮打印数字的Solidity库
  • RareSkills：gas优化指南
  • Vyper 编译流程指南
  • Ape-roll：Ape Framework 的 weiroll（操作链接/脚本语言）客户端
  • ApeWorX cHaOSneT（beta）：付费测试网服务，使用机器人模拟活动
  • Heimdall-rs v0.5.3：提高解码准确性，并添加了帮助解码非标准字节大小交易的标志
  • 隐私游乐场 playgrounds （维基）：链上游戏的隐藏信息机制

安全

  • Stake：主网和侧链热钱包被盗 $4100 万；FBI 指责 Lazarus
  • GMBL Computer 在 Arbitrum 上通过推荐系统发生了~$100 万的攻击
  • FloorDAO 通过 backlog 的 rebase 操作进行了40 ETH 的攻击

生态系统

  • 隐私池的论文（作者包括 Vitalik 和 Ameen Soleimani）：用户可以证明符合监管要求，而不需要透露他们的全部交易历史
  • ERC4337（账户抽象）的拨款受益项目
  • 黑客马拉松项目：ETHWarsaw和Ethcon Korea

链上统计

  • Gas手续费（通过ultrasound.money）：7.8 至 88.9 gwei，平均为 15.7 gwei
    • 目前发行量通缩的gas为 20.4 gwei
    • 本周净发行量为 3.6k ETH
  • ETHUSD：1615-1653 美元，当前为 1635 美元
  • ETHBTC：0.0627-0.0636，当前为 0.0631，反转价格（ETH 超越 BTC 试纸）在约 0.16 左右

应用层的值得注意的事项

  • USDC 在 Optimism 和 Base 上原生发行
  • Kain 提议Synthetix 尝试扩张，在 Base 上使用 ETH 作为抵押品
  • Kevin Owocki回归 Gitcoin
  • Nouns DAO fork #0已开始（fork 的解释：使 20%以上的人可以一起退出他们在财政库中的份额）
  • TryENS：创建一个临时的 ENS 子域名，48 小时后过期
  • Hashmarks：需要使用物理雕刻刷新的逐渐消失的 SVG NFTs
  • Vaults.FYI：为你的资产找到最佳收益（无论是否有农场激励）
